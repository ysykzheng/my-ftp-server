# my-ftp-server

```
usage: java -jar my-ftp-server-*-jar-with-dependencies.jar
       [-h] [-P PORT] [-u USER] [-p PASSWORD] -d DIR

simple ftp server

named arguments:
  -h, --help             show this help message and exit
  -P PORT, --port PORT   ftp port (default: 21)
  -u USER, --user USER   username (default: admin)
  -p PASSWORD, --password PASSWORD
                         password (default: admin)
  -d DIR, --dir DIR      static file dir
```
